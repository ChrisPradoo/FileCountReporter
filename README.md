# File Count Reporter

## Overview
File Count Reporter is a Windows Forms application that analyzes a directory on the user's system and reports file and folder statistics. The application supports both basic directory scanning and full recursive scanning through all nested subdirectories.

## Features

### Version 1: Basic Directory Scan
- Accepts a folder path entered manually or selected through a folder browser dialog.
- Counts all files located directly inside the selected directory.
- Counts all subfolders located directly inside the selected directory.
- Displays results in the application interface.

### Version 2: Recursive Scan
- Traverses all nested subdirectories.
- Counts every file found within the entire directory tree.
- Provides a total recursive file count in addition to the Version 1 results.

## How to Use
1. Launch the application.
2. Enter a folder path manually or select one using the Browse button.
3. Click the Scan button to analyze the directory.
4. View the results displayed in the output fields:
   - Total files (top-level only)
   - Total folders (top-level only)
   - Total files including all subfolders (recursive)

## Running the Project in Visual Studio
1. Open the solution file (.sln) in Visual Studio.
2. Restore NuGet packages if prompted.
3. Build the solution using **Build > Build Solution**.
4. Run the application using **Start Debugging** or **Start Without Debugging**.
